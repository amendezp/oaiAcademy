Create a DCF model

Move into ChatGPT for Sheets or Excel. Use the historical financial statements, country data, and monthly regional data to create the DCF model live instead of starting from a prebuilt template.

Workflow

Turn the finance story into an auditable DCF model.

This is where ChatGPT for Sheets or Excel shines: workbook structure, formulas, editable assumptions, scenarios, valuation outputs, and documentation.

Inputs

What to use

  • The story readout and DCF questions from Workflow 1
  • Historical P&L, balance sheet, and cash flow files
  • Country and regional performance data for growth and margin context
  • An open workbook in Google Sheets or Excel
Outputs

What the model should create

  • Five-year free cash flow forecast with editable assumptions
  • Base, upside, and downside scenarios
  • DCF valuation, net debt bridge, and sensitivity table
  • Source, rationale, confidence, and human review fields

Step by step

Build the model live

  1. Step 1

    Open a blank workbook

    Use Google Sheets or Excel with ChatGPT enabled so the model can be created in the working file.

  2. Step 2

    Import the ChillNest files

    Bring in the historical statements, country data, and monthly regional data. Keep each source easy to inspect.

  3. Step 3

    Paste the prompt below

    Ask ChatGPT to create the model, not just explain how to build one.

  4. Step 4

    Review the workbook

    Confirm outputs use formulas, assumptions are editable, and review fields are visible before moving on.

Copy prompt

Create the DCF Model

Create a simple 5-year DCF model in this workbook using the ChillNest files.

Include:
- historical summary from the P&L, balance sheet, and cash flow statement
- editable assumptions for growth, margin, opex, working capital, capex, tax, discount rate, and terminal growth
- base, upside, and downside forecast
- free cash flow, terminal value, enterprise value, net debt, and implied equity value
- sensitivity table for discount rate and terminal growth
- country priority: Invest, Test, Hold, or Pause

Use formulas for outputs. Do not hardcode forecast or valuation results.

Add source, rationale, confidence, and human review status for key assumptions.
