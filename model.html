DCF Model Creation

Create a DCF model in this workbook using the ChillNest company profile, historical P&L, balance sheet, cash flow statement, country performance data, and monthly regional performance data.

Workflow: DCF Model Creation.

Build the model from scratch. Include:
- historical statement summary: revenue, gross margin, EBITDA, EBIT, net income, working capital, capex, and free cash flow
- forecast assumptions section for revenue growth, gross margin, operating expense, tax rate, working capital, capex, and terminal growth
- base, upside, and downside revenue forecast
- regional monthly trend and seasonality check to inform near-term revenue assumptions
- country performance bridge to explain near-term growth quality
- free cash flow forecast for 5 years
- terminal value
- enterprise value
- net debt adjustment using cash and debt from the balance sheet
- implied equity value
- sensitivity table for discount rate and terminal growth
- investment priority by country: Invest, Test, Hold, or Pause

Create editable assumption inputs:
- weather_demand_lift
- marketing_efficiency_adjustment
- return_rate_adjustment
- logistics_cost_adjustment
- FX_margin_adjustment
- inflation_or_interest_rate_cost_adjustment
- regional_trend_adjustment
- free_cash_flow_conversion_adjustment
- revenue_growth_by_year
- gross_margin_by_year
- opex_as_pct_of_revenue
- working_capital_as_pct_of_revenue
- capex_as_pct_of_revenue
- tax_rate
- discount_rate
- terminal_growth_rate

Use formulas for calculated outputs. Make the model easy for Finance to audit.

Add a brief model documentation section that explains each formula in plain English.

Clearly label which outputs are historical, forecast, valuation, country-level, regional, and total company.
