    Create the DCF Model

    Create a lightweight 5-year DCF workbook for ChillNest using the monthly regional performance file, historical statements, and the Workflow 1 finance story.

Keep it simple and demo-ready. Create only these tabs:
- Source Summary: short summary of the files and key historical numbers
- Assumptions To Review: 6-8 important assumptions with current value, rationale, source, confidence, and review status
- Simple Forecast: 5-year revenue, gross margin, operating expenses, tax, capex, working capital, and free cash flow
- DCF Output: discount rate, terminal growth, enterprise value, net debt, and implied equity value

Use formulas for the forecast and valuation outputs. Do not create sensitivity tables, scenario tabs, charts, or a dashboard yet.

Make the Assumptions To Review tab easy to inspect because we will validate it in the next workflow.
  4. Step 4

    Review the workbook

    Confirm outputs use formulas, assumptions are editable, and review fields are visible before moving on.
