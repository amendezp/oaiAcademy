Create the DCF Model

Create a simple 5-year DCF model in this workbook using the ChillNest files.

Include:
- historical summary from the P&L, balance sheet, and cash flow statement
- editable assumptions for growth, margin, opex, working capital, capex, tax, discount rate, and terminal growth
- base, upside, and downside forecast
- free cash flow, terminal value, enterprise value, net debt, and implied equity value
- sensitivity table for discount rate and terminal growth
- country priority: Invest, Test, Hold, or Pause

Use formulas for outputs. Do not hardcode forecast or valuation results.

Add source, rationale, confidence, and human review status for key assumptions.
