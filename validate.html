Step 3 of 5

Validate assumptions with research

Move back into ChatGPT for this workflow. Validate the highest-impact assumptions with sourced research, then decide what should be applied to the workbook.

Inputs
  • The DCF model summary and assumption table from Workflow 2
  • Source files or exports that explain the current assumptions
  • ChatGPT with Research/Search enabled
  • Internal country, regional, and cash flow context
Outputs
  • A validation table with old value, recommended value, and as-of date
  • Source links and rationale for material updates
  • Confidence and human review status for each key assumption
  • A clear list of approved workbook updates for Workflow 4

  3. Step 3

    Paste the prompt below

    Ask ChatGPT for a validation table and a clear recommendation, not a workbook edit.

    Copy prompt

    Validate Assumptions

    Use the DCF summary, assumptions, and source files to validate the key assumptions.

Use Research/Search, or an approved finance source, to check:
- heat or weather demand
- FX and margin pressure
- inflation or interest-rate pressure
- regional trend durability
- free cash flow conversion
- discount rate and terminal growth

Do not edit the workbook. Create a validation table with assumption, current value, recommended value, source, as-of date, confidence, rationale, and human review status.

End with the exact changes Finance should apply in the workbook, plus a favorable / unfavorable summary of likely model impact.
  4. Step 4

    Approve what moves forward

    Decide which recommendations should be applied to the workbook in the next workflow.

Workflow

Validate assumptions with sourced research and visible judgment.

The point is not to let research autopilot the model. The point is to use ChatGPT to prepare sourced recommendations before any spreadsheet changes are made.

Human review moment: A source link is not the same as approval. Use the confidence and review-status fields to show where Finance still needs to make a judgment call.
