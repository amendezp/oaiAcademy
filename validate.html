Step 3 of 5

Validate assumptions and apply Finance judgment

Keep working in the same workbook. First, validate the assumptions with sources. Then, refine the model using the language and thresholds Finance teams use every day.

Use connectors carefully

Research can help validate weather, FX, inflation, or interest-rate assumptions, but externally sourced values still need human review.

Add team judgment

Finance house style turns a model into a reviewable work product: materiality, favorable or unfavorable language, and clear decisions needed.

Prompt 1

Validate + Refresh Assumptions

Use ChatGPT for Sheets to validate and refresh the key assumptions in this DCF.

Use the Research/Search connector, or an approved finance-data connector, to check:
- heat or weather demand
- FX and margin pressure
- inflation or interest-rate pressure
- regional trend durability
- free cash flow conversion
- discount rate and terminal growth

Update only assumption and review fields. Do not overwrite formulas.

For each update, add source link, as-of date, old value, new value, confidence, human review status, and rationale.

Recalculate the model and summarize what changed.
Prompt 2

Make It Finance-Ready

Make this model finance-ready.

Apply these rules:
- use favorable / unfavorable language
- separate timing issues from recurring issues
- distinguish actuals, assumptions, and externally validated assumptions
- classify country drivers as demand, pricing, returns, logistics, FX, inventory, or execution
- replace generic commentary with the specific driver and business implication

Add concise leadership commentary, open review items, and decisions needed.
Human review moment: A source link is not the same as approval. Use the confidence and review-status fields to show where Finance still needs to make a judgment call.
