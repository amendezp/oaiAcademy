Workflow 3 of 5

Validate assumptions in the workbook

Stay inside ChatGPT for Google Sheets or Excel. First explain what is in each tab, then validate the assumptions and add clickable source links next to the rationale.

Inputs
  • The DCF workbook from Workflow 2
  • Assumption, rationale, and review-status fields in the workbook
  • The Workflow 1 finance story, brief, and open questions
  • ChatGPT for Google Sheets or Excel
Outputs
  • A plain-English explanation of what each workbook tab contains
  • Validated assumptions with recommended updates where needed
  • A clickable external source link column next to each rationale
  • Confidence, as-of date, and human review status for each assumption

Step by step

Run the workbook validation

0 of 2 completed
  1. Step 1

    Explain each tab

    Open the workbook. In Google Sheets, click Extensions. In Excel, click Plugins. Open ChatGPT, then run the prompt below.

    Copy prompt

    Explain Workbook Tabs

    Give me a quick overview of this workbook and each tab's content.
  2. Step 2

    Validate assumptions and add source links

    Ask ChatGPT to review the assumptions tab and add external source links in the right column.

    Copy prompt

    Validate Assumptions

    Review the @assumptions in the assumptions tab. Make sure all of them make sense and add a link to an external data source supporting each claim in column M.

Workflow

Use the workbook assistant to inspect tabs and validate assumptions.

The goal is to keep validation close to the model: understand the workbook structure, check the assumptions, and make source-backed updates visible in the sheet.

Human review moment: A clickable source link is not the same as approval. Open a few sources, check the as-of dates, and use the review-status fields to show where Finance still needs to decide.
