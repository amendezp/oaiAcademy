Workflow

Refresh assumptions with sourced research and visible judgment.

The point is not to let research autopilot the model. The point is to make assumptions, source links, confidence, and human review status visible in the workbook.

Inputs

What to use

  • The DCF workbook created in Workflow 2
  • Editable assumption and review fields
  • ChatGPT for Sheets or Excel with Research/Search
  • Internal country, regional, and cash flow context
Outputs

What changes

  • Updated assumption values with old value, new value, and as-of date
  • Source links and rationale for material updates
  • Confidence and human review status for each key assumption
  • A favorable / unfavorable summary of model impact

Copy prompt

Validate + Refresh Assumptions

Use ChatGPT for Sheets to validate and refresh the key assumptions in this DCF.

Use the Research/Search connector, or an approved finance-data connector, to check:
- heat or weather demand
- FX and margin pressure
- inflation or interest-rate pressure
- regional trend durability
- free cash flow conversion
- discount rate and terminal growth

Update only assumption and review fields. Do not overwrite formulas.

For each update, add source link, as-of date, old value, new value, confidence, human review status, and rationale.

Recalculate the model and summarize what changed using favorable / unfavorable language.
Human review moment: A source link is not the same as approval. Use the confidence and review-status fields to show where Finance still needs to make a judgment call.
